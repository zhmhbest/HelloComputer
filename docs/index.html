ARP

是根据IP地址获取MAC地址的一个TCP/IP协议。ARP协议工作的前提是默认了其所在的网络是一个善良的网络，每台主机在向网络中发送应答信号时都是使用的真实身份（否则会产生ARP欺诈）。

概念

协议端口（Port）

能唯一的表示确定主机下某一进程的整数（取值范围[0,65535][0,65535][0,65535]）。分为服务端口（取值范围[0,49151][0,49151][0,49151]）和客户端口（取值范围[49152,65535][49152,65535][49152,65535]）。

服务端口又分为熟知端口（取值范围[0,1023][0,1023][0,1023]）和登记端口（取值范围[1024,49151][1024,49151][1024,49151]）。

客户端口仅在客户进程运行时才动态选择。

套接字（Socket）

网络中不同主机上的应用进程之间进行双向通信的端点的抽象。

Socket=(Host,Port)Socket = (Host, Port)Socket=(Host,Port)

区别 http https
